# AssessForge - Automated Question Paper Generator

A full-stack web application for automated question paper generation with role-based access control (Admin & Lecturer).

## Technology Stack

- **Frontend**: React.js + Vite + Tailwind CSS
- **Backend**: Node.js with Express.js
- **Database**: MongoDB Atlas with Mongoose ODM
- **API Integration**: Google Gemini Flash 2.0 API
- **Document Generation**: PDFKit for PDF, docx library for Word documents
- **Authentication**: JWT tokens with bcrypt password hashing
- **File Handling**: Multer for file uploads

## Setup Guide

### Prerequisites

- Node.js (v16 or higher)
- MongoDB Atlas account
- Google Gemini API key

### Backend Setup

1. Navigate to the backend directory:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file in the backend directory:
```env
PORT=5000
MONGODB_URI=your_mongodb_atlas_connection_string
JWT_SECRET=your_jwt_secret_key_here
GEMINI_API_KEY=your_google_gemini_api_key_here
NODE_ENV=development
```

4. Create the `uploads` directory:
```bash
mkdir uploads
mkdir uploads/papers
```

5. Start the backend server:
```bash
npm run dev
```

The backend will run on `http://localhost:5000`

### Frontend Setup

1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The frontend will run on `http://localhost:3000`

## Features

### Authentication
- User registration with role selection (Admin/Lecturer)
- Login with JWT token authentication
- Protected routes based on user roles

### Admin Dashboard
- Add/Remove approved lecturer emails
- View all question papers sent by lecturers
- Manage college details (logo, name, address, department)
- Add/Delete subjects

### Lecturer Dashboard
- Add subjects and modules (from admin-added subjects)
- Generate questions using Google Gemini AI
- Create question papers with custom templates
- View unused questions from previous papers
- View and manage generated papers
- Send papers to admin for review

## Database Collections

1. **Users** - User accounts with authentication
2. **ApprovedLecturers** - Approved lecturer emails
3. **CollegeDetails** - College information
4. **Subjects** - Subject list added by admin
5. **Modules** - Modules added by lecturers for subjects
6. **GeneratedQuestions** - Questions generated by AI
7. **QuestionPapers** - Generated question papers
8. **UnusedQuestions** - Unused questions text files

## API Endpoints

### Authentication
- `POST /api/auth/register` - Register new user
- `POST /api/auth/login` - Login user
- `GET /api/auth/me` - Get current user

### Admin Routes
- `POST /api/admin/add-lecturer` - Add approved lecturer
- `GET /api/admin/lecturers` - Get all approved lecturers
- `DELETE /api/admin/lecturers/:id` - Remove lecturer
- `GET /api/admin/college-details` - Get college details
- `PUT /api/admin/college-details` - Update college details
- `POST /api/admin/subjects` - Add subject
- `GET /api/admin/subjects` - Get all subjects
- `DELETE /api/admin/subjects/:id` - Delete subject
- `GET /api/admin/question-papers` - Get all papers sent to admin
- `DELETE /api/admin/question-papers/:id` - Delete paper

### Lecturer Routes
- `GET /api/lecturer/subjects` - Get all subjects
- `POST /api/lecturer/modules` - Add module
- `GET /api/lecturer/modules/:subjectId` - Get modules for subject
- `POST /api/lecturer/generate-questions` - Generate questions using AI
- `POST /api/lecturer/save-question` - Save generated question
- `GET /api/lecturer/available-questions` - Get available questions
- `POST /api/lecturer/create-paper` - Create question paper
- `GET /api/lecturer/unused-questions` - Get unused questions
- `GET /api/lecturer/my-papers` - Get lecturer's papers
- `PUT /api/lecturer/my-papers/:id/send` - Send paper to admin
- `DELETE /api/lecturer/my-papers/:id` - Delete paper

## Notes

- Make sure MongoDB Atlas allows connections from your IP address
- Google Gemini API key is required for question generation
- JWT tokens expire after 7 days
- File uploads are limited to 5MB for logos
- Question papers are generated in both PDF and DOCX formats
